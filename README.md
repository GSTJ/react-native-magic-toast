# react-native-magic-toast

a beautiful imperative toast you can call from anywhere

## Installation

```sh
npm install react-native-magic-toast
```

## Usage

```js
import { multiply } from "react-native-magic-toast";

// ...

const result = await multiply(3, 7);
```
